1. What is a junction?
A Windows junction is a link that redirects one folder location to another.  
It allows your Downloads folder to point to another drive without copying files.

Example:  
Downloads/Resume â†’ G:\MyDrive\Resume  
Note: Files are physically stored on G:, not on C:.

2. How do I connect Google Drive or another cloud drive?
1. Install the official desktop sync app (Google Drive, OneDrive, etc.).  
2. Ensure the cloud storage appears as a local drive on your system.  
3. Create a junction from your Downloads subfolder to the mounted drive location.

Once mounted, routing works like any other drive.

** IMPORTANT NOTES
    1. Junction setup is required for external or cloud routing.  
    2. Deleting the junction does NOT delete files stored on the target drive.  
    3. The extension does not upload, copy, or access file contents. It only redirects download paths using Chromeâ€™s downloads API.
